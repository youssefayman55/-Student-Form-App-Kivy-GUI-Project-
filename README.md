# -Student-Form-App-Kivy-GUI-Project-
A modern desktop form application built using Python and Kivy that allows users to enter and submit student information through an interactive UI.

🚀 Project Description

This project is a GUI-based student form system that demonstrates how to build structured applications using Kivy.

It allows users to:

1- Enter student information

2- Submit data via a button

3- Display results in the console

4- Use a modern styled dark UI

🛠️ Tech Stack:

1- Python 🐍

2- Kivy Framework 📱

3- Kivy Graphics (Canvas UI styling)

📦 Installation:

Install Kivy before running the project:

pip install kivy

▶️ How to Run:
1- Save the script as main.py
2- Run the application: python main.py

🎯 Features:
📝 Student Form Inputs

The app collects:

1- Student ID

2- Student Name

3- Gender

4- Country

5- Academic Year

6- Marks

🎨 Modern UI Design:

1- Dark themed interface

2- Styled input fields

3- Grid-based layout

4- Colored labels and buttons

🖱️ Interactive Button:

Submit button triggers data processing

Displays entered data in console

🧠 Concepts Used:

1- GUI development with Kivy

2- Object-Oriented Programming (OOP)

3- Event handling (button click)

4- Dynamic UI creation

5- Layout management (GridLayout)

6- Canvas customization (background styling)

💡 Future Improvements:

1- 💾 Save data to database (SQLite / Firebase)

2- 📄 Export form data to Excel or CSV

3- 🖥️ Add form validation (required fields)

4- 📊 Show submitted data in UI table

5- 🌐 Convert to mobile app (Android APK)
